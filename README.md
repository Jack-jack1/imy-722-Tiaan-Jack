# Hexadecimal TDD Calculator Course

Welcome to the Hexadecimal TDD Calculator Course repository! This project serves as an online course package designed to teach intermediate web development techniques and strategies, specifically focusing on Test-Driven Development (TDD).

## Project Overview
This repository contains the codebase and tutorial branches for building an online calculator application using a Test-Driven Development model. 

The calculator performs basic arithmetic functions (addition, subtraction, multiplication, and division) strictly on a set of hexadecimal numbers. 

**Application Constraints:**
* Accepts inputs up to 2 digits.
* Returns answers up to 4 digits.
* Does not return negative answers or answers with decimal places.

## Course Structure
This repository is broken down into four distinct sections, with dedicated branches for each phase of the project:

### 1. Identifying Functional Requirements & Setting up the Repository
* Defining features and categorizing inputs/outputs.
* Setting up the GitHub repository.
* Automating tests with GitHub Actions.

### 2. Writing Tests & Drafting Code
* Creating initial test cases and writing basic units.
* Iteratively refining tests and functions to meet all functional requirements.

### 3. Testing the GUI
* Building a mobile-friendly frontend with separate buttons for characters and operations.
* Writing tests to ensure GUI components function correctly.

### 4. Porting to Containers & Cloud Services
* Containerizing the application using Docker.
* Deploying the containerized application to AWS.

## How to Navigate This Repository
* **Branches:** Use the Git branches to navigate between different stages of the tutorial. The branches keep track of the progression of the assignment.
* **Video Tutorials:** Each section is accompanied by short, insightful videos (2-5 minutes) explaining the concepts. Please refer to the designated learning platform or provided drive for the video links.
